



# 07 Acceptance criteria

## 1. Purpose and scope

Defines the acceptance criteria used to judge PoC completion.

This specification references:

* [07-poc/07-acceptance-criteria.md](../../../07-poc/07-acceptance-criteria.md)
* [POC-ACCEPTANCE.md](../../../../docs-build/hybrid/POC-ACCEPTANCE.md)
* [TRACEABILITY.md](../../../../docs-build/hybrid/TRACEABILITY.md)

## 2. Responsibilities and boundaries

This specification is responsible for:

* Declaring which criteria must be demonstrably satisfied.
* Defining canonical AC IDs/titles/details used by generated acceptance ledgers.

This specification does not cover:

* Implementation details of acceptance tests.
