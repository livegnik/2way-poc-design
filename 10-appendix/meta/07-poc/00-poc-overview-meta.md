## 2. Responsibilities and boundaries

This specification is responsible for:

* Defining the PoC documentation set and how it is used.
* Declaring the PoC scope boundaries.
* Enumerating the PoC app domains (contact list, messaging, social, market).
* Declaring PoC terminology distinctions between `Market app`, `Market apps`, and `Marketplace`.
* Declaring requirement ID anchor ranges used by generated PoC app coverage references.

This specification does not cover:

* Protocol details beyond cross-references.
* Implementation details in code.
