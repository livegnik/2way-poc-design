This specification is responsible for:

* Declaring the PoC goals and guarantees.
* Declaring explicit non-goals.

This specification does not cover:

* Detailed build or run instructions.
* Acceptance criteria (see [07-acceptance-criteria.md](../../../07-poc/07-acceptance-criteria.md)).
