



# 05 Messaging flow

## 1. Purpose and scope

Defines the end-to-end flow for the PoC messaging app: creating conversations, sending messages, and reading inbox data.

This specification references:

* [06-flows/05-messaging-flow.md](../../../06-flows/05-messaging-flow.md)
* [POC-APPS.md](../../../../POC-APPS.md)
* [01-protocol/02-object-model.md](../../../01-protocol/02-object-model.md)

## 2. Responsibilities and boundaries

This specification is responsible for:

* Defining the required graph objects for messaging flows.
* Declaring capability checks and ACL enforcement for messaging.

This specification does not cover:

* User interface messaging behavior.
* End-to-end encryption UX or delivery guarantees beyond the protocol.
